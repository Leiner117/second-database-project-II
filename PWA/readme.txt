Aplicación QR PWA
Este proyecto consta de una QR PWA (Progressive Web App) y una aplicación móvil React Native. La QR PWA permite a los usuarios generar códigos QR ingresando su número de identificación y un código. La aplicación móvil React Native proporciona una interfaz de usuario para crear códigos QR.

Instalación
Para ejecutar la aplicación móvil QR PWA y React Native, siga estos pasos

Clona el repositorio.
Navegue hasta el qrdirectorio de QR PWA y el expodirectorio de la aplicación móvil React Native.
Instale dependencias ejecutando npm installo yarn install.
Inicie el servidor de desarrollo usando npm starto yarn start.

Reaccionar aplicación móvil nativa
La aplicación móvil React Native proporciona una interfaz de usuario para generar códigos QR. 
Los usuarios pueden ingresar su número de identificación y un código, y la aplicación generará 
un código QR que podrá ser escaneado por otros dispositivos.

Uso
Ingrese su número de identificación y un código en los campos de entrada.
Presione el botón "Crear QR" para generar el código QR.
El código QR generado se mostrará en la pantalla.
